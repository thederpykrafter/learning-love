# love

## Run project

### from terminal
```bash
love $project_dir
```

### from nvim plugin

| Keymap | Function |
| --- | --- |
| run_project | `<leader>vv` |
| stop project | `<leader>vs` |

## make .love file
```bash
# in project dir
zip -9 -r learning-love.love .
```

## run .love file on android
```bash
termux-open learning-love.love
```
